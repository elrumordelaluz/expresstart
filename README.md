# Expresstart

Starter files for basic [Express](https://expressjs.com/) Server.

- Reloading using `nodemon`
- Using ES6 in the source
- Compiled by `babel@7`

```
# clone the repo
git clone https://github.com/elrumordelaluz/expresstart.git my-express-server
```

```
# install dependencies
yarn
```

```
# dev
yarn start
```

```
# prod
yarn build
yarn serve
```
